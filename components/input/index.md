## Devider API

| 属性       | 说明     | 类型      | 默认值    |
| ---------- | -------- | --------- | --------- |
| `placeholder`     | 输入框中未输入时默认显示的文字 | `string`  | `空` |
| `modelValue` | 输入框输入内容绑定的值   | `string` | `空`   |
| `type` | 输入框的类型    | `string` | `text`   |
| `disabled` | 是否禁用输入框    | `boolean` | `false`   |
