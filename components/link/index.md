## Link API

| 属性       | 说明                                                                                              | 类型      | 默认值    |
| ---------- | ------------------------------------------------------------------------------------------------- | --------- | --------- |
| `type`     | 链接的类型，可以是`primary`、`success` | `string`  | `default` |
| `disabled` | 控制链接是否处于禁用状态，禁用状态下按钮不可点击，常通过变灰等视觉样式体现                        | `boolean` | `false`   |
| `href` | 链接的路径    | `string` | `空`   |
