# Radio 单选框

在一组备选项中进行单选

## 基础使用

单选框不应该有太多的可选项， 如果你有很多的可选项你应该使用选择框而不是单选框。

<demo vue="../../example/radio/base.vue"></demo>

## 禁用状态

<code>disabled</code>属性可以用来控制单选框的禁用状态.

<demo vue="../../example/radio/disabled.vue"></demo>


## Radio API

### Radio 属性

| 属性       | 说明     | 类型      | 默认值    |
| ---------- | -------- | --------- | --------- |
| `modelValue`     | 绑定值 | `string`  | `空` |
