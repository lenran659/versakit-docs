# Switch 开关

表示两种相互对立的状态间的切换，多用于触发「开/关」。

## 基础用法

绑定 <code>v-model</code>到一个<code>Boolean</code>类型的变量。

<demo vue="../../example/switch/base.vue"></demo>


## Switch API

| 属性       | 说明     | 类型      | 默认值    |
| ---------- | -------- | --------- | --------- |
| `v-model`     | 绑定值 | `boolean`  | `false` |
