## 基本使用

<p>通过使用ver-icon来调用SVG图标，并且通过size调整图标的尺寸，通过color改变图标的颜色。</p>

## Icons API

### Icons 属性

| 属性       | 说明     | 类型      | 默认值    |
| ---------- | -------- | --------- | --------- |
| `name`     | 图标名 | `string`  | `空` |
| `size`    |  图标尺寸  | `number` | `14`   |
| `color`    |  图标颜色  | `number` | `#000`   |
