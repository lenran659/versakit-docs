# Drawer 抽屉

有些时候, <code>Dialog</code> 组件并不满足我们的需求, 比如你的表单很长, 亦或是你需要临时展示一些文档, <code>Drawer</code> 拥有和 <code>Dialog</code> 几乎相同的 API, 在 UI 上带来不一样的体验.

## 基础使用

呼出一个临时的侧边栏, 可以从多个方向呼出

<p>
绑定 <code>v-model:value</code>到一个<code>Boolean</code>类型的变量。
</p>
